# Mandelbrot SDL2 Rust

A mandelbrot test with rust and SDL2.

## Running
With rust installed just run (recommended use release option for better performance): 

``` shell
cargo run --release
```

## Keys
| key        | description              |
|------------|--------------------------|
| a          | zoom in                  |
| s          | zoom out                 |
| arrow keys | move the cartesian plane |
| r          | reset mandelbrot         |
| esc        | exit application         |
